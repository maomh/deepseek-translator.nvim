INTRODUCTION                                     *deepseek-translator-intro*

DeepSeek Translator is a Neovim plugin that provides translation and word
lookup functionality using the DeepSeek API.

Features:
  • Word lookup with phonetics, definitions, synonyms, and examples
  • Text translation to Chinese
  • Floating window display
  • Asynchronous API requests

==============================================================================
REQUIREMENTS                              *deepseek-translator-requirements*

  • Neovim >= 0.8.0
  • plenary.nvim (https://github.com/nvim-lua/plenary.nvim)
  • DeepSeek API key

==============================================================================
INSTALLATION                              *deepseek-translator-installation*

Using lazy.nvim: >lua
    {
      'maomh/deepseek-translator.nvim',
      dependencies = { 'nvim-lua/plenary.nvim' },
      config = function()
        require('deepseek-translator').setup({
          api_key = 'your-deepseek-api-key',
        })
      end,
    }
<

==============================================================================
CONFIGURATION                            *deepseek-translator-configuration*

Setup function: >lua
    require('deepseek-translator').setup({
      api_key = '',
      api_base = 'https://api.deepseek.com/v1/chat/completions',
      model = 'deepseek-chat',
      timeout = 30000,
      max_tokens = 2000,
      window = {
        width = 80,
        height = 20,
        border = 'rounded',
      },
    })
<

Options:~
    {api_key}       string  DeepSeek API key (default: DEEPSEEK_API_KEY env)
    {api_base}      string  API endpoint URL
    {model}         string  Model name (default: 'deepseek-chat')
    {timeout}       number  Request timeout in ms (default: 30000)
    {max_tokens}    number  Max response tokens (default: 2000)
    {window}        table   Window configuration
      {width}       number  Window width (default: 80)
      {height}      number  Window height (default: 20)
      {border}      string  Border style (default: 'rounded')

==============================================================================
USAGE                                            *deepseek-translator-usage*

Normal mode:
  Place cursor on a word and press <leader>t to look up the word.

Visual mode:
  Select text and press <leader>t to translate it.

Floating window:
  Press 'q' or <Esc> to close the result window.

==============================================================================
COMMANDS                                      *deepseek-translator-commands*

:DeepseekTranslateWord                          *:DeepseekTranslateWord*
    Look up the word under the cursor.

:DeepseekTranslateSelection                *:DeepseekTranslateSelection*
    Translate the visually selected text.

==============================================================================
MAPPINGS                                      *deepseek-translator-mappings*

Default mappings:

    {mode}  {lhs}           {rhs}
    n       <leader>t       <Plug>(deepseek-translate-word)
    v       <leader>t       <Plug>(deepseek-translate-selection)

Plug mappings:~

<Plug>(deepseek-translate-word)         *<Plug>(deepseek-translate-word)*
    Look up word under cursor (normal mode).

<Plug>(deepseek-translate-selection) *<Plug>(deepseek-translate-selection)*
    Translate selection (visual mode).

Custom mappings example: >vim
    nmap <leader>tw <Plug>(deepseek-translate-word)
    vmap <leader>ts <Plug>(deepseek-translate-selection)
<
